# Real-time Chat App

A simple real-time chat application built with **Node.js**, **Express**, and **Socket.io**.

## 🚀 Features
- Real-time messaging using WebSockets
- Express.js backend with **Socket.io**
- Basic frontend with **HTML, CSS, and JavaScript**
